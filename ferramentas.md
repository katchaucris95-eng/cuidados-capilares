# CSS: Quais propriedades estão sendo usadas?
margin: Define a margem externa do elemento, ou seja, o espaço transparente entre a borda do elemento e outros elementos adjacentes. O valor 0 remove qualquer margem padrão que o navegador possa aplicar.

padding: Define o preenchimento interno do elemento, ou seja, o espaço entre o conteúdo do elemento e sua própria borda. O valor 0 remove qualquer preenchimento padrão.

box-sizing: Altera a forma como a largura e a altura totais de um elemento são calculadas. O valor border-box instrui o navegador a incluir o padding e a border (borda) dentro da largura (width) e altura (height) especificadas para o elemento. Isso facilita o design e o layout, pois o tamanho total do elemento permanece o mesmo, independentemente do preenchimento ou borda adicionados.

background-color: Define a cor de fundo do elemento. 

background-image: Define uma imagem como plano de fundo do elemento. 

color: Define a cor do texto. 

text-align: Define o alinhamento horizontal do conteúdo do elemento. Neste caso, o valor center centraliza o texto.

display: flex;: Define o elemento como um contêiner flexível (flex container). Isso permite que você organize seus elementos filhos (itens flexíveis) em layouts flexíveis, controlando seu alinhamento, direção e ordem com facilidade.

gap: 4px;: Define o tamanho do espaço (ou "gutter") entre as linhas e colunas dos itens dentro do contêiner flexível (ou grid). Neste caso, um espaço de 4 pixels será aplicado entre os itens.

list-style: none;: Remove os marcadores de lista padrão (como os pontos ou números) de um elemento de lista (geralmente <ul> ou <ol>). Isso é comumente usado para estilizar listas de navegação onde os marcadores visuais não são desejados.

padding: define uma a distância entre o conteúdo de um elemento e suas bordas.
